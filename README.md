# Backend of an Ecommerce Platform

A backend part of an ecommerce application using the MERN Stack (MongoDB, Express.js, React.js and Node.js).

## Features
- RESTful API
- JWT Authentication (HTTP-Only Cookies)
- Role-Based Authorization
- Product Management APIs
- Order Management APIs
- Cart Management APIs
- Admin Panel
- MongoDB with Mongoose
- Request Validation
- Password Hashing (bcryptjs)
- Error Handling Middleware
- Secure Cookie & CORS Configuration
- AI Product Description Generation using Hugging Face for Admin
- Modular MVC Architecture
